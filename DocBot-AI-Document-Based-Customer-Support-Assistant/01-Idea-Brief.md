# DocBot AI – Document-Based Customer Support Assistant

## Problem Statement
Customers often spend too much time searching through long PDF documents for specific information.

## Target User
Customers who need quick answers from a company's documentation.

## Value Proposition
Users can ask questions in natural language and receive answers based only on the provided document.

## MVP Scope
- Chat interface
- Fixed PDF stored in backend
- AI answers only from the document
- Loading indicator
- Friendly error handling

## Top Risks
1. AI hallucination
2. PDF parsing issues
3. Large document size

## Biggest Assumption
The PDF contains searchable text and can be parsed successfully.
